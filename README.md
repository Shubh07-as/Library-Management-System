# Library-Management-System

"Console-based library management system in Java with book tracking and status management"

## Features
- Add new books
- Display all books
- Issue books to students
- Accept returned books
- Status tracking (Available/Issued)

## Technologies Used
- Java
- Scanner class for user input
- Arrays for data management

## How to Run
1. Compile: `javac LibraryManagement.java`
2. Run: `java LibraryManagement`
